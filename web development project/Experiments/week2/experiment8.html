Types of cursor

Cursor change by hover

Cursor property mentions the type of cursor to be displayed when pointing on a given element. It serves to be very useful while designing a website.

Hover over the words to change cursor.

Pointer : Shows link.

Move: Shows something to be moved.

Wait: busy.

Not Allowed : can not respond.

Cross Hair: cross hair cursor.

Help : help is available.

Right Arrow: edge of given box .(right)
