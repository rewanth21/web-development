Planets


Planet Layout:Lets start from the top of the page


CSS:
All the pages except the homepage use the same style. The top of the page consists on a menu bar with all the planets, the solar system, Asteroid belt and Beyond Nav bars. Clicking on either of these bars will lead to a different page.

Then comes a slideshow for each page with four images changing at a linear time.All of this is placed in a container with CSS effects. The next major effects are the descriptions to the right of the webpage where hovering over the objects displays a small box with a brief description of each object.

Scrolling further down will show a brief video about each object. The videos has been taken from youtube where the embedded html code is already provided by clicking on the share option and then clicking on the embed option. Clicking on the "back to top" button will take user user back to the top of the webpage.


Content:
The content has a layout. Starting with Mass, Diameter, Orbit period and distance from the Sun, Surface temperature etc. Then a brief history and some of the unique aspects about the planet.

After this comes the Atmospheric composition and internal structure of the planet. Then the orbit and Rotation of the planet and finally some interesting facts about the planet followed by the video.
